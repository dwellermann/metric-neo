# Dirty Notes:

Wenn eine neue MEssung durchgefürht wird, muss bei der Munition darafu hingewiesen werden eine Chargen nummer einzugeben.
Dies ist wichtig, um sicherzustellen, dass alle Messungen korrekt zugeordnet werden können und die Rückverfolgbarkeit gewährleistet ist. Ohne die Eingabe der Chargennummer könnten Verwechslungen auftreten, die die Integrität der Messdaten beeinträchtigen.
Vorallem dann wenn die Munition schon einmal verwendet wurde aber eine andere Chargennummer hat.
